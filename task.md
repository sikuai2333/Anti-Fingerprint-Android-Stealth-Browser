## 1) 代理节点与订阅能力改造（高优先级）

### 目标
删除当前硬编码第三方代理节点，改为可维护的订阅机制。

### 任务
- [ ] 删除 `MainActivity` 中硬编码 `tg://proxy?...` 节点建议列表。
- [ ] 新增“订阅管理”能力，支持用户手动添加/编辑/删除订阅源。
- [ ] 订阅格式支持：
  - [ ] Clash
  - [ ] V2Ray
- [ ] 订阅导入方式支持：
  - [ ] 二维码导入
  - [ ] 本地配置文件导入
  - [ ] URL 导入
- [ ] 增加“手动更新订阅”入口与更新状态提示（成功/失败/最后更新时间）。
- [ ] UI 与交互可参考：v2rayNG、FlClash（仅作交互参考，不直接复制实现）。

### 验收标准
- [ ] 代码中不再出现硬编码第三方代理节点。
- [ ] 用户可在 UI 内完成订阅新增、更新与选择。
- [ ] 订阅异常（超时/格式错误/无节点）有明确提示与降级处理。
